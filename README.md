# 🍪 Panimii Bakery - E-commerce

Plataforma de repostería artesanal especializada en roles y galletas, diseñada con un enfoque minimalista y optimizada para despliegue en entornos cPanel.

## 🚀 Tecnologías
- **Backend:** Django 5.x
- **Base de Datos:** MySQL (Producción) / SQLite (Desarrollo)
- **Frontend:** Tailwind CSS + [design.md](DESIGN.md)
- **Procesamiento de Imagen:** Pillow (Compresión automática a WebP)

## 🎨 Sistema de Diseño
Este proyecto utiliza la especificación `DESIGN.md` de Google Labs para mantener la consistencia visual. Puedes encontrar los tokens de diseño y la guía de estilo en el archivo [DESIGN.md](./DESIGN.md).
